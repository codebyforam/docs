# Authentication Flow

## Purpose

Authentication system manages:
- User login/signup
- Admin authentication
- Session management
- Role-based access

## Authentication Methods

### User Authentication
- Phone Number OTP Login
- Optional Google Login

### Admin Authentication
- Email + Password

## User App Flow

Open App
→ Splash Screen
→ Login Screen
→ OTP Verification
→ User Dashboard

## Admin Flow

Admin Login
→ Email/Password
→ Admin Dashboard
