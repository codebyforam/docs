# Admin Access Structure

## Admin Panel Modules

- Dashboard
- Bookings
- Services
- Slots
- Events
- Users
- Notifications
- Content
- Donations
- Settings

## Access Rules

### Super Admin
Can access all modules.

### Admin
Can access:
- Bookings
- Services
- Slots
- Events
- Users
- Notifications
- Content

### Content Manager
Can access:
- Content
- Gallery
- Banners
