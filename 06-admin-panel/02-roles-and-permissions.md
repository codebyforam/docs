# Roles & Permissions

## Super Admin

Permissions:
- Full system access
- Manage admins
- Manage bookings
- Manage users
- Manage donations
- Manage settings

---

## Admin

Permissions:
- Approve/reject bookings
- Manage events
- Upload content
- Send notifications

---

## Content Manager

Permissions:
- Upload pravachans
- Upload images/videos
- Manage banners
- Manage gallery

---

## Volunteer Manager

Permissions:
- Manage volunteers
- Assign seva
- Approve volunteer requests

---

## User

Permissions:
- Create booking
- View booking history
- View events
- Receive notifications
